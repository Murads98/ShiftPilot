How It Works:
  1. Select a template containing your weekly shift pattern
  2. Choose the date range for which you want to generate shifts
  3. The template will be applied to each week in the range
  4. Monday shifts from the template will be created on all Mondays, Tuesday shifts on all Tuesdays, etc.
  5. If a shift already exists for a specific date and shift type, it will be skipped (unless overwrite is enabled)
Note: This will create actual shift records in your database. Make sure to review the generated shifts and assign employees before publishing the schedule.
